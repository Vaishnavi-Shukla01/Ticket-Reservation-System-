# Ticket-Reservation-System-
A JavaFX-based Ticket Reservation System with role-based login, route management, booking &amp; cancellation features, and persistent storage. Includes a rule-based chatbot for handling user queries related to routes, seat availability, and booking.

# How to Run the Project
- Prerequisites

Make sure you have the following installed:

- Java JDK 17 or above
- Maven
Any IDE (VS Code / IntelliJ) or terminal
1. Clone the Repository
  git clone https://github.com/<your-username>/Ticket-Reservation-System.git
  cd Ticket-Reservation-System
2. Run the Application
   Using Maven (Recommended)
   mvn javafx:run
